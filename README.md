# PPOCeleste
a mod about a PPO agent in celeste modding

## Installation

### Prerequisites

- [CelesteTAS](https://github.com/EverestAPI/CelesteTAS)
- [Everest](https://everestapi.github.io/)
- [MonoMod](https://github.com/MonoMod/MonoMod)

### Installation Steps

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/PPOCeleste.git
    cd PPOCeleste
    ```

2. Build the project:
    ```bash
    dotnet build
    ```

3. Copy the built DLL to your Celeste Mods folder:
    ```bash
    cp bin/Debug/netstandard2.0/PPOCeleste.dll /path/to/Celeste/Mods/
    ```

## Usage

1. Launch Celeste with Everest.
2. Load your desired save file.
3. Start the game and observe the PPO agent in action.
